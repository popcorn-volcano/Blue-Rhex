## 🔑 Authentication Help

If you get "Please provide a password" during push:

**Option A: Use GitHub Token (Recommended)**
1. Go to: https://github.com/settings/tokens/new
2. Generate a new token (Personal Access Token)
3. Select: `repo` scope
4. Copy the token
5. When prompted for password during `git push`, paste the token

**Option B: Setup SSH Keys**
- Instructions: https://docs.github.com/en/authentication/connecting-to-github-with-ssh

**Option C: Configure Git Credentials**
```powershell
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```
